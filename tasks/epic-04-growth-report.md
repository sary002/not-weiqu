## 1. Epic 概述

### 1.1 目标
让用户在不内卷的前提下，看见「比昨天的自己更稳」的变化；提供周 / 月 / 季报 + 日记摘要。

### 1.2 范围
- 进度条目（bucket + count）
- 关键里程碑
- 周 / 月 / 季报生成
- 日记摘要
- 复盘页 UI（可关闭）
- 数据不内卷红线

### 1.3 不在范围
- 排行榜 / 等级 / 打卡
- 社区
- 公开分享

### 1.4 关键指标
| 指标 | 目标 |
| --- | --- |
| 报告生成 P95 | ≤ 2.0s |
| 摘要生成 P95 | ≤ 2.5s |
| 「打卡天数」类 | 0 |
| 用户关闭率 | ≤ 30% |
| 触动率 | ≥ 70% |

## 11. v2.0 同步说明（2026-06-16）

> 主体范围（进度 / 里程碑 / 周月季报 / 日记摘要）保留。
> v2.0 调整：复盘从独立页 → 收纳进「我 → 进度」子页；指标体系重做。

### 11.1 指标体系（v2.0 重做）

| v1.0 指标 | v2.0 指标 | 备注 |
| --- | --- | --- |
| 月度曲线 | **区段掌握度**（§1–§5） | 5 段独立进度 |
| 30 日"说不"次数 | **练过的瞬间 N 次** | 累计，**不**按"天数" |
| 打卡天数 | **温和连击 N 天** | 缺失静默归零 |
| 排名 / 排行榜 | **无** | v2.0 拒绝 |
| 完整体验 N 次 | **覆盖的微技能 N / 17** | 横跨 5 段 |

### 11.2 文案红线（v2.0 强约束）

| 场景 | ❌ 禁用 | ✅ 替换为 |
| --- | --- | --- |
| 进度总览 | "你已经坚持 23 天了！" | "练过 23 次" |
| 月报 | "挑战成功！🎉" | （无） |
| 周报 | "别忘了明天练习" | "想练就练" |
| 季报 | "排名上升 5 位" | （无） |
| 通知 | "你今天还没练完" | （不显示） |

### 11.3 复盘页位置（v2.0 调整）
- v1.0：独立底部 Tab
