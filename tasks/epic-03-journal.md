## 1. Epic 概述

### 1.1 目标
让用户能把「用过的 / 管用的话术」保存下来，反复看、随时调取；演练过程可暂停 / 保存。

### 1.2 范围
- 「我的剧本」保存 / 编辑 / 删除
- 剧本标签（职场 / 家庭 / 朋友 / 亲密）
- 演练过程暂停 / 保存
- 一键分享（脱敏）

### 1.3 不在范围
- 社区 / 树洞
- 跨平台同步（先 web）
- 真实人对接

### 1.4 关键指标
| 指标 | 目标 |
| --- | --- |
| 剧本保存成功率 | ≥ 95% |
| 剧本查询 P99 | ≤ 200ms |
| 分享脱敏 | 100% |

## 10. v2.0 同步说明（2026-06-16）

> 主体范围（保存 / 编辑 / 删除 / 标签 / 暂停 / 分享脱敏）保留。v2.0 调整：

### 10.1 形态调整
- 「我的剧本」从 v1.0 子页提升为 v2.0 底部 4 Tab 之一（与「今日」「自由对话」「我」并列）
- 暂停 / 保存与 v2.0 的 12 步演练绑定（步骤 6 暂停 → 草稿；3 天后归档）

### 10.2 草稿机制（v2.0 新增）

| 状态 | 时长 | 计入累计？ |
| --- | --- | --- |
| 草稿（durable） | ≤ 3 天 | 是（继续完成即计入） |
| 过往的尝试 | > 3 天 | **否**（不写入"放弃"叙事） |
| 完成 | 完成 12 步 | 是 |

> 草稿超时**不**显示"你放弃了一次练习"等内疚文案。

### 10.3 拒绝清单自检
- 剧本列表**不**展示"坚持 X 天"等
- 剧本保存提示**不**用"打卡成功"
- 分享脱敏**不**带可识别第三方（v1.0 已实现，v2.0 沿用）
