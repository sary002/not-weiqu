# /status — 项目状态快照

> 一眼看清「我们现在在哪」。

## 触发
`/status` 或 `/status <feature>`

## 行为
1. 扫 `plans/` 的所有计划 → 状态（计划中 / 设计中 / 实施中 / 验收中 / 完成）
2. 扫 `tasks/` 的所有任务 → 阻塞 / 进行 / 完成
3. 扫 `docs/decisions/` 的 ADR 数量与最近一次
4. 输出：
   - 整体进度（一句话）
   - 阶段分布
   - 阻塞清单
   - 风险登记
   - 下一步建议

## 产出
仅打印到对话，不写文件（除非用户要求快照）。

## 适用 Agent
Coordinator
