# 用户画像（Personas）

> 文档状态：v0.1 草稿 · 负责人：PM · 评审：KE + PE

> 我们不创造新的心理分类，只把已经存在的「善良的难处」说清楚。

## 主画像：阿瑶（职场讨好者）

- **年龄**：26
- **性别**：女
- **职业**：互联网公司运营
- **性格**：善良、敏感、善于察言观色
- **痛点**：
  - 同事甩活过来不敢拒绝
  - 加班到深夜心里委屈但嘴上说「没事」
  - 朋友圈发个动态都要想半天会不会让别人不舒服
- **她想要的**：
  - 一份能照着说的「拒绝话术」
  - 一次不会被评判的「演练」
  - 一个不会用「你应该…」训她的对话对象
- **她害怕的**：
  - 被评价为「太脆弱」「想太多」
  - 拒绝后关系破裂
  - 暴露自己「其实很委屈」

## 副画像：阿磊（亲密关系讨好者）

- **年龄**：32
- **性别**：男
- **职业**：工程师
- **痛点**：
  - 伴侣冷暴力时不断道歉
  - 父母催婚时无法表达真实想法
  - 朋友借钱时不敢要回来
- **他想要的**：
  - 不被「嫌弃」的安全环境
  - 可被拆解为「一句开场 → 一个动作」的步骤
  - 情绪低谷时能快速找到一个出口

## 副画像：小敏（家庭关系讨好者）

- **年龄**：40+
- **职业**：教师
- **痛点**：
  - 大家庭里永远在「成全别人」
  - 兄弟姐妹、父母、孩子、亲戚的需求优先级永远排在自己前面
  - 一旦表达需求就被指责「自私」
- **她想要的**：
  - 重新认识「自私」的边界
  - 一点一点的小练习，而不是一夜之间翻脸

## 反画像（不是我们的用户）
- 想要「话术包搞定一切」的人 → 我们不卖话术
- 想要「攻击 / 报复第三方」的人 → 我们不教恨
- 已经在精神科治疗急性期的人 → 我们会转介

## 共同需求
| 维度 | 共同点 |
| --- | --- |
| 触发场景 | 职场 / 家庭 / 亲密关系 / 友谊 |
| 情绪低谷 | 深夜、周末独处、刚被回绝后 |
| 求助路径 | 先自己扛 → 朋友 → 短视频 → 树洞 → 搜资料 → 求助专业 |
| 接受 AI 的门槛 | 「不像机器人」「不说教」「可以打断」 |
| 敏感点 | 评价、催促、暴露、被贴标签 |
