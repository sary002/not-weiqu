# ADR-003: UI 视觉语言 v2（圆润仙人掌 + 鼠尾草绿）

- **状态**：Proposed
- **日期**：2026-06-17
- **决策者**：PM（主）+ FE（联合）+ 用户（产品方向）
- **影响模块**：全局视觉 / 品牌识别 / Tailwind 主题 / 字体 / 图标 / 暗色模式
- **触发**：用户对当前页面设计反馈「页面设计也不太好」，给出 3 个具体品牌锚点

---

## 背景

### 用户反馈（2026-06-17）

> "页面设计也不太好"
> "Logo: 圆润仙人掌 🌵"
> "配色: 鼠尾草绿"
> "Slogan: 帮善良的人学会有锋芒"

### 现状问题

| 问题 | 表现 | 根因 |
| --- | --- | --- |
| **视觉温度不足** | Tailwind 默认色 + Shadcn 默认组件 = 工具感 | 没有专属视觉语言 |
| **品牌识别弱** | 用户记不住产品 | 配色 / 图标 / 字体都用默认值 |
| **微交互粗糙** | hover / focus 缺少细腻反馈 | 套默认 transition |
| **信息层级扁平** | 顶部 + 卡片 + 底部 Tab 视觉权重平均 | 没有设计 Token 体系 |

### 关键约束

- **不评判、不催促、不替代专业咨询**（CLAUDE.md §1）
- **红线 1-8**（CLAUDE.md §5）：不用红色警告 / 不用"🔥" / 不用打卡天数
- **可访问性**：WCAG 2.1 AA 对比度
- **多端**：Web + 移动端（PWA，详见 ADR-004）

---

## 决策

### 三个品牌锚点（用户给定 · 不可改）

| 项 | 决策 |
| --- | --- |
| **Logo** | 圆润仙人掌 🌵（圆角柱状 + 顶部小红果 / 花） |
| **主色** | 鼠尾草绿（sage green） |
| **Slogan** | 帮善良的人学会有锋芒 |

下面所有规范围绕这 3 个锚点展开。

---

| `sage-300` | `#A8C190` | 禁用态 / 占位 |

| Token | Hex | 用途 |
| --- | --- | --- |

| `sunset` | `#E8B47C` | 沙漠夕阳 / 进度环激活态 |

**禁用色**

| ❌ 禁用 | 原因 |

| 纯红 `#FF0000` 系列 | 红色 = 警告 / 评判工具，违反红线 1 |

| 高饱和小麦黄 | 工具感 |
| "🔥" 表情 | 红字，违反红线 3 |

### 2. Logo 设计 · 圆润仙人掌

**形态规范**

```
        ●           ← 顶部小红果（cactus-flower 色，圆点直径 = 柱身 1/3）
       ╱│╲
      ╱ │ ╲          ← 主体柱身，圆角矩形（corner radius = 柱宽 1/4）
     │  │  │
     │  │  │
     │  │  │
     │  │  │
     ╲ │ ╱
      ╲│╱
       ▼             ← 底部圆角（无尖锐）
```

**设计要素**

| 项 | 规范 |
| --- | --- |
| 主形 | 圆角矩形柱身（圆角 = 柱宽的 25%） |
| 高度比 | 柱身高度 : 柱身宽度 = 2.5 : 1 |
| 顶部红果 | 实心圆点，颜色 `cactus-flower`，直径 = 柱宽 30% |
| 侧面小刺（可选） | 极小圆点 / 短线，浅色 `sage-300`，不锐利 |
| 底部 | 圆角收尾，无尖锐线条 |
| 描边 | 不用 1px 硬描边，用 2px 圆角 + 同色填充 |
| 多尺寸适配 | ≥ 64px 显示完整刺点；< 32px 仅显示柱身 + 红果 |

**Logo 用法**

| 场景 | 规格 |
| --- | --- |
| 主页 hero | 完整版（带刺点），≥ 120px |
| BottomNav 简化 | 仅柱身 + 红果，≤ 32px |
| Favicon | 仅柱身 + 红果，16/32/48px |
| PWA 启动屏 | 完整版 + slogan，512x512 |
| 加载动画 | 呼吸感缩放（详见 §4） |

**禁用**

- ❌ 加尖锐刺（与"圆润"原则冲突）
- ❌ 加阴影投射（避免沉重）
- ❌ 旋转 / 翻转（避免工具感）
- ❌ 配喜庆红 / 警告红

### 3. 字体系统

**中文字体**

| 用途 | 字体 | 字重 |
| --- | --- | --- |
| 标题 / Slogan | 思源宋体（Source Han Serif SC） | 400 / 600 |
| 正文 | 思源黑体（Source Han Sans SC） | 400 / 500 |
| 按钮 / 标签 | 思源黑体 | 500 |

**英文字体**

| 用途 | 字体 |
| --- | --- |
| 标题 | iA Writer Quattro / IBM Plex Serif |
| 正文 | Inter / IBM Plex Sans |

**字号体系（rem-based）**

| Token | Size | 用途 |
| --- | --- | --- |
| `text-xs` | 0.75rem (12px) | 标签 / 提示 |
| `text-sm` | 0.875rem (14px) | 辅助文字 |
| `text-base` | 1rem (16px) | 正文 |
| `text-lg` | 1.125rem (18px) | 卡片标题 |
| `text-xl` | 1.25rem (20px) | 段落标题 |
| `text-2xl` | 1.5rem (24px) | 区块标题 |
| `text-3xl` | 1.875rem (30px) | 页面主标题 |
| `text-4xl` | 2.25rem (36px) | Slogan |

**行高 / 字间距**

- 正文：line-height 1.7（更舒展，避免压迫感）
- 标题：line-height 1.3
- 字间距：默认 letter-spacing 0；标题 -0.01em（紧凑但不挤压）

### 4. 微动效（灵魂所在）

**核心原则**：**不抢戏、不打扰、像呼吸一样**

| 场景 | 动效 | 时长 | 缓动 |
| --- | --- | --- | --- |
| 进入练习 | 缓慢淡入 + 轻微上移（8px） | 400ms | ease-out |
| 完成反馈 | 节点状态翻转 + 文字轻轻浮现 | 600ms | cubic-bezier(0.25, 0.1, 0.25, 1) |
| 页面切换 | 横向淡入（不用滑动） | 300ms | ease-out |
| 按钮按下 | 轻微下沉（0.5px）+ 颜色变深 | 100ms | ease-in-out |
| Logo 加载 | 呼吸感缩放（scale 1 → 1.05 → 1） | 2000ms | ease-in-out infinite |
| 进度条 | 平滑过渡（不用 step） | 800ms | ease-out |

**禁用动效**

| ❌ 禁用 | 原因 |
| --- | --- |
| 旋转 / 翻转 | 工具感 |
| 弹跳 / 弹性曲线 | 喧闹、不稳重 |
| 🎉 彩带 / 烟花 | 评判胜利感，违反红线 1 |
| 抖动 / 闪烁 | 违反红线 7 |
| 红色感叹号 | 警告感，违反红线 1 |

**全局开关**

```css
/* 全局尊重 prefers-reduced-motion */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 5. 暗色模式（深夜陪伴）

**触发条件**

- 系统设置 `prefers-color-scheme: dark`
- 用户在设置页手动切换（默认 follow system）
- 22:00 ~ 06:00 自动推荐（不强切，弹提示"夜深了，要不要切到夜间模式？[稍后][切换]"，**不二次确认**）

**配色映射**

| 亮色 Token | 暗色 Token | 说明 |
| --- | --- | --- |
| `warm-50` 背景 | `sage-900` | 深绿底 + 暖灰文字 |
| `warm-700` 文字 | `warm-100` | 暖米文字 |
| `sage-400` 主色 | `sage-300` | 暗色下主色更亮 |
| 卡片 `warm-100` | `sage-800` | 卡片背景更深 |

**不变**

- Logo（仙人掌在任何背景下都清晰）
- 主品牌色（鼠尾草绿在暗色下饱和度调高 10%）
- 红色点缀（cactus-flower 仍可用，暗色下稍微提亮）

### 6. 图标系统

**替换方案**

| 当前 | 替换为 | 原因 |
| --- | --- | --- |
| Lucide Icons（线性 1.5px） | **Phosphor Icons**（圆润 + 双线） | 工具感 → 陪伴感 |
| 默认 | Heroicons（圆润） | 部分场景 |

**图标风格规范**

- 描边 1.5px（不用 2px 硬描边）
- 圆角端点（不用方形端点）
- 双线 + 实心填充混合（Phosphor 的 "duotone" 风格）
- 不用 emoji 替代（emoji 风格不统一）

**关键图标**

| 用途 | 图标 |
| --- | --- |
| 今日（home） | 仙人掌简化版（特殊处理） |
| 自由对话 | 对话气泡（圆角） |
| 我的剧本 | 书本（圆角） |
| 我（profile） | 圆形头像轮廓 |
| 危机热线 | 电话（圆角） |
| 温和连击 | 火焰的弱化版（**不**用 🔥）—— 用叶子 / 小苗 |

### 7. 圆角 / 阴影 / 间距系统

**圆角**

| Token | Value | 用途 |
| --- | --- | --- |
| `rounded-sm` | 4px | 标签 / 小按钮 |
| `rounded-md` | 8px | 卡片 / 输入框 |
| `rounded-lg` | 12px | 大卡片 / 弹窗 |
| `rounded-xl` | 16px | 底部 Tab |
| `rounded-2xl` | 24px | 兜底页容器 |
| `rounded-full` | 9999px | 头像 / 圆形按钮 |

**阴影（柔和、无硬投影）**

| Token | Value | 用途 |
| --- | --- | --- |
| `shadow-sm` | `0 1px 2px rgba(74, 92, 57, 0.05)` | 卡片轻浮起 |
| `shadow-md` | `0 4px 8px rgba(74, 92, 57, 0.08)` | 弹窗 |
| `shadow-lg` | `0 8px 24px rgba(74, 92, 57, 0.12)` | 兜底页 |

**阴影色**：用 sage-700（深鼠尾草绿）作 alpha，**不用纯黑**（避免硬投影）。

**间距（4px 基准）**

- `1=4px` / `2=8px` / `3=12px` / `4=16px` / `6=24px` / `8=32px` / `12=48px` / `16=64px`

### 8. 品牌声音 / 文案语气

**Slogan 用法**

| 场景 | 用法 |
| --- | --- |
| 启动屏 | 居中显示，slogan + Logo |
| 主页底部 | 弱化显示，作为签名 |
| 关于页 | 完整呈现 + 一句话解读（"我们陪你练出属于自己的边界"） |

**禁用词（基于现有红线）**

- ❌ 加油 / 坚持 / 挺住 / 忍一忍
- ❌ 你没事吧 / 想开点
- ❌ 必须 / 应该 / 你连这都
- ❌ 失去 / 丢失 / 再不来就 / 你快没了
- ❌ 打卡 N 天 / 连续 N 天

**推荐词**

- ✅ 试试 / 慢慢来 / 我在听
- ✅ 你刚才说的 / 我注意到
- ✅ 今天先到这里 / 休息一下
- ✅ 你愿意 / 你可以

## 后果

### 得到

- ✅ 专属品牌识别（仙人掌 + 鼠尾草绿 = 强记忆点）
- ✅ 视觉温度提升（从工具感 → 陪伴感）
- ✅ 微动效增加"灵魂"（呼吸感 vs 弹跳）
- ✅ 暗色模式（夜间陪伴）
- ✅ 设计 Token 体系（后续扩展可复用）

### 失去

- ❌ Tailwind 默认色板（需要定制）
- ❌ Lucide Icons（部分需替换）
- ❌ 现有部分组件需要重写（GlobalHeader / BottomNav / SkillNode / CrisisBanner）

### 风险

| ID | 风险 | 等级 | 缓解 |
| --- | --- | --- | --- |
| R-1 | Logo 设计周期长 | P1 | 用简单 SVG 骨架（柱身 + 红果）先上线，刺点后续迭代 |
| R-2 | 字体加载慢 | P2 | 用 `next/font` + display: swap + subset（中文字体只取常用字） |
| R-3 | 暗色模式对比度不达标 | P1 | 跑 WCAG AA 自动化测试；所有色对 ≥ 4.5:1 |
| R-4 | 重写现有组件影响其他任务 | P0 | 先做 UI 改造 → 再做 13 任务实施；或并行但用 design token 隔离 |

### 回滚条件

- 用户对 ADR-003 任一决策明确反对 → 切回 Shadcn 默认 + 重新评审
- 实施后发现核心组件（SkillNode / CrisisBanner）改动 > 50% → 拆分为 v2.1 重审
