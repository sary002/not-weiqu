## 3. 背景与机会

### 3.1 行业背景
讨好型人格（People-Pleasing Personality）在中国年轻互联网用户中具有高发趋势，叠加文化中「以和为贵」的社会期待，使该群体在职场、家庭、亲密关系中长期处于「压抑需求 → 累积委屈 → 关系失衡」的循环中。

公开数据点（KE 调研）：
- 知乎、小红书「讨好型人格」相关话题年阅读量在亿级。
- 心理学类公众号、播客订阅量年增长 ≥ 30%。
- 「树洞」类应用与「情绪记录」类应用 DAU 持续增长。

### 3.2 现有方案的不足
| 现有形态 | 不足 |
| --- | --- |
| 心理学公众号 / 课程 | 缺乏可演练的反馈环 |
| 树洞 / 社区 | 容易变成吐槽区，缺少可执行的下一步 |
| 心理测评 / 标签 | 容易强化自我标签化，引发焦虑 |
| 真人对接（咨询师） | 门槛高、成本高、频次不可持续 |
| 通用 AI 助手 | 缺乏人格设计，容易说教 / 评价 |

### 3.3 我们的机会
**用「人格化 AI 陪练」+「可演练脚本」+「危机兜底」三件套，做一款**可重复、低门槛、有边界的成长陪伴产品**。

## 5. 产品愿景与价值观

> 详见 `docs/product/vision.md`，本节摘录产品级约束。

- **愿景**：帮善良的人学会有锋芒。
- **三年北极星**：≥100 万用户、NPS≥60、危机转介成功率 100%。
- **价值观优先级（冲突时按此排序）**：
  ```
  用户安全 > 用户长期成长 > 商业可持续 > 短期增长
  ```

## 6. 核心价值主张

| 价值 | 形态 | 用户感知 |
| --- | --- | --- |
| 陪练，不是说教 | AI 角色扮演 + 反馈 | 「像有个朋友陪我练」 |
| 锋芒不是攻击 | 平静表达脚本 | 「原来我也可以这样说」 |
| 成长可被看见 | 进度记录（不内卷） | 「原来我比昨天更稳了」 |
| 危机有兜底 | 危机话术 + 转介 | 「最难的时候有人接住我」 |
| 隐私可托付 | 脱敏 + 可删除 | 「我说的话不会变成伤害我的证据」 |

## 7. 边界五层模型（产品视角）

> 由 PM 维护，与 KE 的「内容五层」对齐。

| 层 | 名称 | 关键动作 | 关键能力 | MVP 优先级 |
| --- | --- | --- | --- | --- |
| L1 | 觉察 | 识别委屈的身体/情绪信号 | 情绪命名、复盘式回顾 | P0 |
| L2 | 命名 | 把情绪与需求说清楚 | 「我需要 / 我不想 / 我愿意」 | P0 |
| L3 | 表达 | 在真实场景演练 | 角色扮演 + 脚本库 | P0 |
| L4 | 兜底 | 被回绝后自我安抚 | 危机话术 + 转介资源 | P0 |
| L5 | 巩固 | 把单次胜利变成习惯 | 进度复盘 + 微习惯 | P1 |

> 每条需求必须挂到这五层之一。

## 11. 范围与边界

### 11.1 MVP 包含
- Onboarding（不审问）
- 今日练习（P0）
- 自由对话（P0）
- 兜底与危机（P0）
- 我的剧本（P0）

### 11.2 MVP 不包含
- 社区 / 树洞
- 真人对接（咨询师）
- 心理测评 / 标签
- 排行榜 / 打卡天数解锁
- 视频 / 直播
- 任何形式的诊断

### 11.3 暂缓
- 多模态（语音、图片）
- 知识小卡自动生成
- 跨平台（先 Web，后 App）
