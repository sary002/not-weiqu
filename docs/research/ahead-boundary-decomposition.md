## 1. Ahead 课程目录(估计)

> 基于公开评测(Forbes Health、Verywell Mind、Healthline)与课程截图描述,**约 30-50 个主题课程**,分为 6 大类(具体数字需核验):

| # | 类别 | 估计课程数(≈) | 与不委屈相关度 |
|---|------|--------------|---------------|
| 1 | 焦虑 / 压力管理 | 8-10 | 中(焦虑是边界退缩的常见并发症) |
| 2 | 自信 / 自我价值 | 5-7 | 高(自我价值是边界表达的内核) |
| 3 | 关系 / 边界 | **3-5** | **极高(直接对标)** |
| 4 | 睡眠 / 放松 | 4-6 | 低 |
| 5 | 习惯 / 目标 | 3-5 | 低 |
| 6 | 正念 / 情绪觉察 | 4-6 | 中(L1 觉察的素材) |

**关键发现**:Ahead 的边界主题课程占比估计 < 10%,**说明这是一个被市场验证但供给不足的细分赛道**。这是不委屈的机会。

## 2. Ahead 边界类课程的结构(估计)

> 一个 5-10 分钟的 Ahead 边界课程通常包含以下结构(基于公开评测的描述与 CBT 通用范式):

### 2.1 课程开场(约 30-60 秒)

- **教练口播**(音频为主):"嗨,今天我们聊聊如何回应同事甩活这件事"
- **情境锚定**:"你先想一想:你最近一次被人甩活是什么时候?"
- **时长控制**:开场不超过 1 分钟,防止用户感到压力

### 2.2 核心内容(约 3-5 分钟)

- 典型结构:**情境 → 思维 → 情绪 → 替代反应**
- 多模态:教练音频 + 文字总结卡片 + 偶尔插图
- 内容深度:**CBT 通用自助方法**(避免具体疗法主张,符合不委屈 `KE-R4`)

### 2.3 互动练习(约 2-3 分钟)

- **Thought Record 卡片**:情境 → 自动想法 → 情绪强度(1-10)→ 替代想法
- **选择题**:在不同回应中选最贴近自己的
- **自由文本**:"用你自己的话写一句回应"

### 2.4 收束(约 30 秒)

- **教练总结**:"今天你看到了自己在 X 情境中的反应模式"
- **1 句提示**:"下次遇到类似情况,记得你还有另一种选择"
- **心情追踪**:"你现在的感受如何?"

### 2.5 课程结束页

- 心情记录打卡
- 推荐下一节课程
- 进度条推进
- 偶尔有"分享到社区"(可选,非强制)

secondary_layer: L1  # 可能跨层

tone_fallback: calm_brief  # 深夜 / 疲惫时

  opening: "<教练开场白,30-60 字>"
  anchor: "<情境锚定提问,30 字内>"
  
main_content:
  - stage_1_fact: "<看到的事实>"
  - stage_2_thought: "<常见自动思维>"
  - stage_3_emotion: "<典型情绪 + 强度>"
  - stage_4_alternative: "<替代反应示范>"

interactive_exercise:
  - type: thought_record
    fields: [situation, auto_thought, emotion_intensity, alternative]
  - type: choice
    question: "<具体问题>"
    options: ["<选项 1>", "<选项 2>", "<选项 3>"]
  - type: free_text
    prompt: "<自由书写引导>"

  summary: "<教练总结,30 字内>"

  - name: "<来源>"
    type: "book|journal|app|other"
    note: "<核验情况>"

risks:
  - "<不适用场景,如 '本内容不适用于长期家暴场景'>"

red_lines:
  - "<该内容不准在 X 场景出现>"
  
audit:
  - date: 2026-06-16
    reviewer: KE
    status: draft

## 5. Ahead → 不委屈的可平移元素 vs 必须原创元素

### 5.1 可平移(借鉴结构与节奏)

| Ahead 元素 | 不委屈对应 | 改动 |
|-----------|-----------|------|
| 5-10 分钟微课 | `12 步演练` 的步骤拆分 | 步数细化,每步 ≤ 30 秒 |
| 教练音频开场 | PE 注入人格(温柔姐姐 / 理性朋友 / 活泼伙伴) | 选教练后才有音频,非全场景默认 |
| Thought Record 卡片 | 步骤 2-5 卡片化输入 | 与 `reply.md` CRIA 输入对齐 |
| 心情追踪 | 收束态的 1-5 标度打卡 | 不显示"昨天 vs 今天"对比 |
| 教练总结 + 1 句提示 | 步骤 12 的"练过了一次"收束 | **去掉感叹号**(竞品研究 §9 UI 规则) |
| 课程进度条 | 技能树 5 段骨架 | **不显示分数**,只显示"来过 N 次" |

### 5.2 必须原创(不照搬)

| Ahead 元素 | 不委屈处理 | 理由 |
|-----------|-----------|------|
| **多教练人格** | 3 个教练 + 用户自选 | 与 Ahead 风格一致,但不委屈需要更多「非评判」保证 |
| **每日课程推荐** | 「今日三件小事」(1 必做 + 2 选做) | 默认 Casual,可降级,避免 Duolingo 式施压 |
| **分享到社区** | **不做**(M2+ 暂缓) | 讨好型人格用户更怕被评判,社区暴露风险高 |
| **进度排行 / 朋友** | **不做** | 竞品研究 §6.1 明确反对 |
| **虚拟货币 / 解锁** | **不做** | 用"温柔的提示"或"教练鼓励卡片"替代 |
| **月度订阅墙** | **弱化付费墙**(M3 决策) | 免费首日 + 付费解锁后**不催促续费** |

### 5.3 必须重新设计的(保留意图,改变形式)

| Ahead 元素 | 不委屈改造 | 改造理由 |
|-----------|-----------|----------|
| **课程结束页的"分享"** | 改为"保存到我的剧本"(本地) | 不外露社交,降低评判风险 |
| **心情曲线的"对比"** | 只显示趋势,不显示对比 | 不委屈 `KE-R6` |
| **每周回顾** | 用"我 → 进度"的"练过的瞬间"展示 | 不显示"坚持 N 天"(竞品研究 §6.1) |
| **推荐下一节** | 默认 0 推荐,用户自选 | 避免 Duolingo 式的「下节课」催促 |

## 6. 不委屈的边界主题差异化优势(基于 Ahead 缺口)

> Ahead 课程结构是「自助式 CBT + 教练音频」,**没有以下三点**:

### 6.1 不委屈独占的 3 个差异化点

1. **角色扮演 + 12 步演练**
   - Ahead 只到「Thought Record」就停了
   - 不委屈让用户**真的说出那句拒绝话** → 步骤 6 选开场白 → 步骤 7 输入说出口的话 → 步骤 8 应对反驳
   - 这是从「想到」到「做到」的关键桥梁

2. **可保存的「我的剧本」**
   - Ahead 没有「沉淀用户已说出口的话」的机制
   - 不委屈的「我的剧本」是用户的**情绪资产**,回看时是「我能做到」的实证
   - 对长期留存关键

3. **危机兜底 + 弱强度对话**
   - Ahead 教练是录音,**无法根据用户当下情绪动态调整**
   - 不委屈的 LLM 是实时对话,**可承接危机信号**
   - 这是 Ahead 永远做不到的(`S-2 危机响应`是 LLM 独有优势)

### 6.2 KE 必须把「差异化」翻译成「内容可生产性」

- 角色扮演 → KE 需要为每节微技能设计 **3-5 个 AI 反驳话术**(步骤 8)
- 我的剧本 → KE 需要设计「剧本」模板(情境 + 说的话 + 当时感受 + 后续感受)
- 危机兜底 → KE + PE 联合维护 `agents/knowledge-engineer/red-lines.md`
