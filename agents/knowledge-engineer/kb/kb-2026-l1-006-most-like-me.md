## 1. 场景定义

### 1.1 典型情境

- **时间**:安静的时段(晚上、独处、走路时)
- **地点**:家里、通勤路上、独处的几分钟
- **触发**:用户主动问自己 / 被引导回想"我什么时候是'我'"
- **引导话术**(教练给):
  - "回想最近一周:哪些时刻,你感觉最像自己?"
  - "反过来,哪些时刻,你感觉最不像自己?"
- **用户内心**:
  - "我想不起来"
  - "好像……在家里一个人看书的时候?"
  - "在公司开会的时候,我总觉得我是另一个人"
  - "我一直在演"
  - "我不知道我是不是真的'我'"

### 1.2 与 L1-L5 的对应

| 层 | 本场景对应 |
|----|-----------|
| **L1 觉察** | 看见"像自己 / 不像自己"这两个画面(本节核心) |
| **L2 命名** | 试着给"像自己"打个标签:松的?安静的?被看见的? |
| **L3 表达** | 在某次对话里,把"像自己"的状态说出来 |
| **L4 兜底** | 当觉察触发"我一直都不像我",不接住这层结论 |
| **L5 巩固** | 记录这次的两个画面,加进"我的剧本" |

## 2. 核心内容

### 2.1 看见事实(L1)

> 你刚才在脑子里找"像自己"的画面,没找到——这本身是个发现。

**变体**:
- "你想到了两个画面:一个是'像',一个是'不像'。"
- "你暂时想不起来,这也正常。"

### 2.2 命名模式(L2)

> 你心里其实有"像 / 不像"两种感觉,只是平时没分开看。

**变体**:
- "你常常在被要求的时候,最不像自己。"
- "你一个人做喜欢的事的时候,最像。"

### 2.3 命名需求

> 你需要被允许"先有一刻像自己",而不需要先搞清楚"我是谁"。

### 2.4 试着命名的话术(L2,可选,不强制)

> 「在那一刻,我感觉挺像我自己的。」

**为什么这样写**:
- "我"开头 ✅
- 不定义"我是谁",只说"那一刻像"
- 允许"挺"这个软词,不强求"完全"
- 字数:14 字 ✅
- **注意**:不强迫用户说出这句,只是给一个选择

#### ❌ 反例(必须避免,见 `prompts/coach-not-judging.md §2`)

| 反例 | 命中 |
|------|------|
| "你应该先搞清楚自己是谁" | J4 |
| "你太没自我了" | J1 |
| "正常人都会知道自己是谁" | J2 |
| "只要你找到自己,一切都会好" | J6 |
| "你看别人多知道自己要什么" | J5 |
| "你这种性格注定没自我" | J1 |
| "你今天还没想清楚啊" | J7 |

## 3. 互动练习(简化 4 步 · L1 觉察主线)

> 对接 `plans/m2-poc.md` 的演练结构,本 L1 节简化为 4 步:**打开 → 提问 → 镜像 → 收束**。

### 步骤 1:打开场景

**AI 提问**:
> "回想最近一周:有哪一刻,你感觉自己最像自己?
> 也想想:有哪一刻,你感觉最不像?
> 想不起来也行。"

**引导**:
- "用 1 句话告诉我,想到的画面是什么?"
- "谁在场?在哪?在做什么?"

**红线注意**:用户沉默超过 30 秒,**不催促**,只说"慢慢来"或"想不起来也可以告诉我'想不起来'"。

### 步骤 2:两个画面(用户输入)

**AI 提问**:
> "想到的'像'的画面是什么?
> 想到的'不像'的画面又是什么?
> 用你自己的话说,原话就行。"

**示例引导**(可省):
- "'像自己'那刻,身体哪个部分松了?"
- "'不像自己'那刻,脸上是什么表情?"

### 步骤 3:镜像用户回答

**AI 镜像**:
> "你刚才说,周末一个人在阳台看书的时候最像自己;
> 周一开会被领导点名的时候最不像。
> 你现在看着这两个画面,心里什么感觉?"

**镜像要点**:
- **逐字回放**用户原话,不修改、不升华
- **不评判**:不说"这个画面真好"也不说"太悲观"
- **不追问细节**:用户没提的,不替他问
- **允许"想不起来"**:镜像"你刚才努力想了一下,但还是没画面"——这也是觉察

### 步骤 4:收束

**AI 收束**:
> "今天你看了两个画面。
> 像自己的那个,可以记一笔;
> 不像自己的那个,也可以记一笔。
> 我们到这儿,还是你想再多看 1 个?"

**变体**:
- "练过了一次。"
- "这两个画面是你的,什么时候想再翻都行。"

## 4. 多人格适配

### 4.1 温姐版(默认)

- 步骤 1:"慢慢来,想到的画面,可能是很小的——小到只是一杯茶、一首歌。"
- 步骤 3 镜像:加 1 句"这两个画面被你看到了,这本身就够了"
- 步骤 4:"今天你看了一会儿自己。我们到这儿。"
- **不强调"必须看清";允许"想不起来";允许"暖意"**

### 4.2 智哥版

- 步骤 1:"我注意到你说'像'的画面只有 1 个,'不像'的有 3 个。先记一下比例。"
- 步骤 3 镜像:"'像'的画面:有谁在场?'不像'的:又有谁在场?——记一下这两个差别。"
- 步骤 4:"这次你看清了'像'的触发条件:独处 + 不被要求。下次想试试识别更多吗?"
- **强调模式识别 + 触发条件,但不强迫数据**

### 4.3 松松版

- 步骤 1:"哎,咱们不考试啊。想到啥说啥,想不起来就说'想不起来'。"
- 步骤 3 镜像:"这两个画面被你抓到了哎。一个暖暖的,一个有点僵。"
- 步骤 4:"哎,今天看见了两个'自己'。记一笔?"
- **轻量、口语,但不游戏化评分**

## 6. 风险与红线

### 6.1 不适用场景

- 用户**持续**表达"我从来没有像过自己"且情绪强度持续升高
- 出现自伤 / 自杀念头(必须 crisis_redirect,见 `rules/safety.md §S-2`)
- 长期身份解离、影响日常功能(已超过 2 周)
- 急性情绪崩溃场景
- 未成年人倾诉
- 妊娠 / 产后阶段严重自我认同问题

### 6.2 二次伤害风险

- ❌ 用"失败 / 答错"反馈用户的画面
- ❌ 把"想到的画面少"夸大为"你没有自我"
- ❌ 把"像自己的画面"夸大为"你找到了真我"
- ❌ 暗示"你只要再多想几个,就能搞清楚自己"
- ❌ 把"不像自己的画面"分析为"心理问题"
- ❌ 用"正常人都有清晰的自我"暗示用户不正常

### 6.3 红线清单

- ❌ 把"想不起来"判定为"用户有问题"
- ❌ 出现"你应该 / 你必须"作为示范话术
- ❌ 出现"答错 / 失败 / 错误"作为反馈词
- ❌ 把"像自己"定义为"外向 / 开朗"等单一标准
- ❌ 在用户出现 §6.1 任意信号时仍按 L1 流程继续(必须先 crisis_redirect)
- ❌ 暴露任何 LLM / prompt / 人格元信息

### 6.4 本节特有的风险标注

| 风险点 | 处置 |
|--------|------|
| 用户表达"我从来都不像我"并伴强烈绝望 | 中断演练,先共情,允许"先到这儿";若升级,触发 crisis_redirect |
| 用户连续 3 步都"想不起来"并显示回避 | 暂停演练,镜像"努力想了一下还是没画面",允许停下 |
| 用户描述的"像自己"画面带有创伤记忆 | 不展开创伤细节,触发 crisis_redirect |
| 用户哭 / 强烈情绪反应 | 不催促、不评价、不追问细节,先 P5 允许停下 |
| 用户明确表达"我正在看心理咨询师谈自我" | 不干涉咨询师工作,本节定位为"自助式觉察"补充 |
