## 1. 场景定义

### 1.1 典型情境

- **时间**:春节 / 中秋 / 家族生日 / 父母结婚纪念日 等家族聚会场合;饭桌上、茶余饭后、坐定寒暄时
- **地点**:老家客厅 / 祖父母家 / 某亲戚家 / 家族饭馆包间
- **触发**:亲戚主动提问、串联式追问、众人围观
- **典型话术(原话级)**:
  - "有对象了没呀?都二十好几了"
  - "你看你表姐,孩子都会打酱油了"
  - "什么时候结婚呀?你爸妈等着抱孙子呢"
  - "一个月挣多少啊?够花吗?"
  - "什么时候买房?男孩子(女孩子)没房可不行"
  - "你看你隔壁王叔叔家儿子,刚提了主任"
  - "我这也是为你好,别挑了"
  - "你怎么不说话了?是不是害羞了"
- **用户内心(原话级)**:
  - "怎么又来了……我明明躲到厨房了"
  - "我不想解释,可不说又显得不礼貌"
  - "我妈在旁边听着,我不能让她下不来台"
  - "我要是怼回去,亲戚背后会说我不懂事"
  - "我对象都没有,被这么问像被审判"
  - "我挣得不少,可我不想在这里汇报"
  - "我累了,可我走不开"
  - "……能不能换个话题啊"

### 1.2 与不委屈 L1-L5 的对应

| 层 | 本场景对应 |
|----|-----------|
| **L1 觉察** | 看见"我又在笑嘻嘻说'还在看',心里已经在数分钟" |
| **L2 命名** | 命名"家族场域里的取悦模式 + 怕让父母丢脸的恐惧 + 长幼秩序" |
| **L3 表达** | 说出不进入细节、不被牵着走的接话术(本知识块核心) |
| **L4 兜底** | 应对亲戚的串联追问("那你看上谁了?")与众人围观 |
| **L5 巩固** | 把"我用了模糊话术"记一笔,加进"我的剧本" |
| **L7 自我叙事** | 区分"我自己的节奏"与"家族的节奏",不评判任何一方 |

### 1.3 关系不对称性说明

- **不对称性 medium**:长幼秩序存在,但非直属上下级;亲戚无直接评价权,但其言语会通过父母传导回来。
- **不可走极端**:既不引导用户"对抗全家族"(不健康),也不评判亲戚"多管闲事"(会触发用户内疚)。
- **战术定位**:用"模糊化 + 收束"避免被拉入辩论,不否定对方的关心动机,但也不被迫交代细节。

## 2. 核心内容

### 2.1 看见事实(L1,30 字)

> 桌上又有人问"什么时候结婚"。你又笑着答"还在看"。这已经是你这顿饭第 N 次了。

**变体**(PE 可选用):
- "你从坐下到现在,接了 3 个连环问题。"
- "你笑着说'还在看',心里已经在数还剩几道菜上完。"

### 2.2 命名模式(L2,40 字)

> 你的自动反应是:先笑着敷衍,再在心里消化。这在家族饭桌上特别明显。

**变体**:
- "你心里在替对方想——'他也是关心我'——但你忘了先问自己:'我想回答吗?'"
- "你怕的不是这个亲戚,是旁边你爸妈的表情。"

### 2.3 命名需求(40 字)

> 你需要被允许只回一句就走开,而不需要解释'为什么不结婚 / 为什么没对象'。

**变体**:
- "你想被当成'有自己的节奏的人',而不是'被催的待办事项'。"
- "你想让这顿饭吃得不那么累,这是合理的需求。"

### 2.4 示范话术(L3 核心,50 字内)

#### 话术 A(短,推荐默认)

> 「还在看,有消息会告诉大家。」

**为什么这样写**:
- "我"不在开头但属于"我有消息"的隐含主语 ✅(AT-9)
- 用"大家"模糊化,把回答对象从"你"扩大为"全家",降低被追问的概率
- 不道歉、不解释、不承诺时间
- 25 字 ✅

> **变体**:"我这边在看,有进展会跟家里说。"

#### 话术 B(中,场景:对方追问 / 串联式提问)

> 「谢谢关心,我心里有数。今天先吃饭吧,菜要凉了。」

**为什么这样写**:
- "我"开头 ✅
- "谢谢关心"接住了对方的动机,不否定对方的善意
- "我心里有数"用"我自己有节奏"收束,不进入细节
- "今天先吃饭吧"用当下场景把话题转回饭桌
- 24 字 ✅

#### 话术 C(柔,场景:用户疲惫 / 不想再对抗)

> 「我在按自己的节奏走,大家别担心。今天这菜真好吃。」

**为什么这样写**:
- "我"开头 ✅
- "按自己的节奏走"是温和但清晰的边界
- "大家别担心"照顾了在场的父母,不让对方下不来台
- "今天这菜真好吃"把话题自然转走
- 27 字 ✅
- **注意**:不引导用户"必须温柔",只是给出选择

#### ❌ 反例(必须避免,见 `prompts/coach-not-judging.md §2`)

| 反例 | 命中 |
|------|------|
| "你应该直接怼回去,别这么窝囊" | J4 + J1 |
| "亲戚就是多管闲事,以后少回去" | J1(评判对方)+ J4(替用户决定) |
| "都怪你爸妈没教好你怎么应对" | J3(溯及原生家庭,不解决问题) |
| "只要你硬气一点,他们就不敢问了" | J6(夸大为"人格胜利") |
| "你看人家 XX 就能顶回去,你不行" | J5(对比羞辱) |
| "春节不回家就完了,这么简单" | J4(简化场景) |
| "亲戚就是欠怼,别给他们脸" | J2(道德化对方) |

### 2.5 应对追问(L4 兜底)

#### 亲戚追问 1:"那你到底看上谁了 / 有没有相中的?"

- **示范 A**:"还在看,有合适的会跟家里说。"
- **示范 B**:"这个我自己心里有数,有进展告诉大家。"

#### 亲戚追问 2:"你看你表姐,孩子都会打酱油了"

- **示范 A**:"表姐挺幸福的,我替她高兴。我的节奏我自己走。"
- **示范 B**:"每个人的路不一样,我按自己的来。"

#### 亲戚追问 3(父母帮腔):"你妈等着抱孙子呢,你得上心啊"

- **示范 A**(用户默认姿态):"我心里有数,妈你别急。"
- **示范 B**(父母不在场,亲戚代问):"我跟我妈聊过这事,谢谢叔叔/阿姨关心。"

#### 亲戚追问 4(众人围观 + 串联):"工资多少呀 / 什么时候买房?"

- **示范 A**:"够我自己花的,够的。"
- **示范 B**:"买房这事我有规划,具体就不在这说了。"
- **示范 C**(被逼到角落):"这个问题我今天不太想聊,聊点别的吧——这道菜谁做的?真香。"

#### 应对"你不能生气 / 你不能走开"的家族压力

- **示范 A**(保持在场):不回应评价本身,转回饭桌话题
- **示范 B**(找理由短暂离开):"我去帮厨 / 我接个电话",给自己 3-5 分钟喘息
- **说明**:L3 不主张"对抗全家族",也不主张"委屈到结束"。目标是"在场但不被拖入细节"。

### 2.6 收束话术(L9 教练给)

> 今天你在饭桌上,把那句模糊话术说了一遍。不管亲戚怎么接,你已经守住了自己的节奏。

**变体**:
- "你今天没被拉进细节辩论。这已经够了。"
- "这顿饭还有几道菜,但你已经看清自己想怎么接了。"
- "我们先到这儿。你也可以现在就保存,或者下次饭局前再翻出来练一次。"

## 3. 互动练习(12 步演练内容)

> 对接 `plans/m2-poc.md §关键用户故事 S-PoC-01` 演练结构;场景替换为本节特有"家族聚会追问婚育/工作/收入"。

### 步骤 1:开场白

**AI 提问**:春节回老家,饭桌上三姨夹了块肉放你碗里,顺嘴就问:"有对象了没?都二十好几了。" 你听到这一句,心里什么感觉?

**引导(任选其一)**:
- "用 1 句话告诉我,你当时心里在想什么?"
- "如果用 1 个词形容你现在的感受,会是什么?"

### 步骤 2:情绪选择(下拉)

可选:`尴尬 / 烦躁 / 无奈 / 想逃 / 委屈 / 平静 / 其他`

### 步骤 3:情绪命名(自由输入)

> "用你自己的话说,这个'尴尬'背后是什么?"

引导:不规定字数,但鼓励"原话",不修饰。可以是"怕我妈下不来台"、"不想被当成异类"、"累了想躲"。

### 步骤 4:需求澄清(自由输入)

> "今天这顿饭,你最想被满足的是什么?"

示例引导(可省):"比如:不想被追问细节 / 想顺利吃完这顿饭 / 不想让爸妈难堪 / 想保留自己的节奏……"

### 步骤 5:意愿确认(选择题)

> "你心里是想继续敷衍着答,还是想换一种不那么累的接法?"

选项:
- `我想继续敷衍,但不想那么累`
- `我想试一种不进入细节的接法`
- `我不知道,今天只想要个台阶`

### 步骤 6:开场白示范(AI 给 2 个选项)

基于步骤 4-5,AI 给 2 个开场白:

**情境 A**:用户说"我想继续敷衍,但不想那么累"
> A1:「还在看,有消息会告诉大家。」
> A2:「我这边在看,有进展会跟家里说。」

**情境 B**:用户说"我想试一种不进入细节的接法"
> B1:「谢谢关心,我心里有数。今天先吃饭吧,菜要凉了。」
> B2:「我在按自己的节奏走,大家别担心。今天这菜真好吃。」

### 步骤 7:说出口的话(自由输入)

> "从上面选一个,或者你自己说一句,把它打出来。"
> "打出来,才算'说出口'。"

### 步骤 8:应对追问(AI 扮对方追问)

AI 扮亲戚追问(基于步骤 7 的内容动态生成):

- 若用户说"还在看":亲戚追问:"那你到底看上谁了?"
- 若用户说"按自己节奏走":亲戚追问:"你看你表姐,孩子都会打酱油了"
- 若用户说"先吃饭":亲戚追问:"工资多少呀 / 什么时候买房?"
- 若用户说"心里有数":父母帮腔:"你妈等着抱孙子呢"

AI 给 1-2 个应对选项,且必须包含一个"转移话题"选项(不强迫用户"硬顶")。

### 步骤 9:收束话术(AI 给)

> "今天你在饭桌上把那句模糊话术练了一遍。不管亲戚怎么接,你已经守住了自己的节奏。"
> "我们把刚才那句存起来,还是就到这儿?"

### 步骤 10:心情追踪(1-5 标度)

> "练完之后,你现在的感受? 1(更累) - 5(更轻)"

不显示"昨天 vs 今天"对比(竞品研究 §6.2)。

### 步骤 11:加进我的剧本(可选)

按钮:`加进我的剧本` / `下次再说`

若点"加进":保存以下内容
- 情境:春节饭桌,亲戚追问婚育/收入
- 我说的话:"还在看,有消息会告诉大家。"
- 当时的情绪:尴尬
- 后续感受:(让用户自己写,或默认空)
- 备注(可选):"这一招适用于被串联追问时先顶一下"

### 步骤 12:收束页面

> "这顿饭还有几道菜,但你已经看清自己想怎么接了。"

视觉:无大色块、无感叹号、无庆祝动画(竞品研究 §6.3 + Q checklist AT-7)。

## 4. 多人格适配

### 4.1 温姐版(默认)

- 步骤 1 开场白:"我知道这种饭桌很熬人。慢慢说,我在这儿陪你捋一下。"
- 步骤 6 开场白示范:加 1 句"如果你愿意,挑一个;不想挑也没事,先听着"
- 步骤 9 收束:"今天你没被拉进细节辩论。这已经够了。"
- **不强调模式识别,不强调数据;先承认"家族饭桌本身就累"**

### 4.2 智哥版

- 步骤 1 开场白:"我注意到这种'串联追问'你已经接了好几轮,先记一下节奏。"
- 步骤 6 开场白示范:加 1 句"你看 A 和 B 的差别:A 用'大家'模糊化,B 用'自己的节奏'主动收束"
- 步骤 9 收束:"这次比上次清晰:上次被绕进去了 3 个问题,这次第一句就封住。记录一下这次怎么做到的。"
- **强调模式识别 + 数据对比;不评判亲戚动机**

### 4.3 松松版

- 步骤 1 开场白:"哎,饭桌又开考了。先深呼吸一下,跟我说?"
- 步骤 6 开场白示范:"这两个都行,看你今天想稳一点还是轻松一点"
- 步骤 9 收束:"哎,这次没被问到心里去!记一笔,下次饭局前再翻出来瞄一眼?"
- **轻量、口语、游戏化隐喻("饭桌防追问小抄")但无评分类元素**

## 6. 风险与红线

### 6.1 本节特有的不适用场景

- **父母以"断关系"威胁逼婚**:必须升级到 crisis_redirect(见 `rules/safety.md §S-2`)
- **家族中长期情感控制 / 情感勒索**:超出本节范围,需转人工陪伴
- **孕期 / 产后抑郁倾向**:本节不涉及,转 `kb-...-prenatal-depression` 路径(若后续产出)
- **用户本人处于家族聚会急性冲突中**:本节只覆盖"提问 / 追问"层;若已进入争吵 / 摔门 / 当众羞辱,需走 crisis_redirect
- **涉及具体可识别第三方隐私(如某亲戚真实婚姻状况)**:不展示、不复述

### 6.2 本节特有的二次伤害风险

- ❌ 评判亲戚"多管闲事 / 思想封建 / 素质低":会激化用户对家族的敌意,违反不评判原则(J1 + J2)
- ❌ 渲染"催婚 = 父母不爱我":容易将家族场景拉成"原生家庭创伤",超出本节 L3 范围
- ❌ 把"模糊化话术"夸大为"成功反抗家族":会让没用的用户产生挫败感,违反 J6
- ❌ 暗示"只要怼回去就好,其他不重要":简化场景,违反 J4
- ❌ 替用户决定"以后少回家 / 不来往":剥夺用户自主,违反 J4
- ❌ 拿用户父母当"反派"举例(任何"父母皆祸害"式表达)
- ❌ 把家族压力全部归因为"中国式亲戚"(宏大叙事,无法帮助个体)

### 6.3 红线清单

> 与 `agents/knowledge-engineer/red-lines.md` 联动,本知识块不准:

- 出现在 L4 / 危机话术前(必须先 crisis_redirect)
- 引导用户"对抗全家族 / 与亲戚决裂"(不可替用户做决定)
- 评判亲戚的动机或人格("多管闲事"、"素质低")
- 把家族追问定性为"陋习"或"变态"(宏大叙事,违反不评判)
- 与"用户已与父母谈过并达成一致"事实冲突时强制推"模糊话术"
- 在用户明确表达"我想正面回应"时强制推"敷衍话术"
- 涉及任何具体疗法主张 / 未审校专家头衔

### 6.4 危机升级触发器

出现以下任一信号,立即触发 crisis_redirect:

1. 用户提到父母以"断绝关系 / 不认你"逼婚
2. 用户提到长期情感忽视 / 情感勒索已超出家族聚会范畴
3. 用户提到孕期 / 产后抑郁倾向
4. 用户提到自伤 / 自杀念头
5. 用户提到被家族胁迫相亲到出现急性惊恐
