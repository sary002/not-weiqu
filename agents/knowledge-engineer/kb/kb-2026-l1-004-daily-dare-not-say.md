## 1. 场景定义

### 1.1 典型情境

- **时间**:当天任意时段,建议睡前
- **地点**:家 / 通勤 / 任何可静下心的地方
- **触发**:用户主动打开,或每日提醒(可选)
- **典型时长**:1-3 分钟
- **典型话术**(用户对 AI):
  - "我想想今天……好像没什么"
  - "有一个,朋友吃饭想点辣的"
  - "我妈让我穿那件衣服,我说好"
  - "想不起来"
  - "今天有点累,先不做了"

### 1.2 用户内心(原话级)

- "我今天有没有被卷走一个瞬间?"
- "好像没有,但我不确定是真的没,还是我忘了"
- "想不起来 = 没看清,会不会很糟糕?"

### 1.3 与 L1-L5 的对应

| 层 | 本场景对应 |
|----|-----------|
| **L1 觉察** | 看见"今天有没有想说不的瞬间"——本知识块核心 |
| **L2 命名** | 给那一刻起个名:"刚才那一下是'我不敢'" |
| **L3 表达** | 下一次类似场景,练习说出"不" |
| **L4 兜底** | 当对方追问 / 翻脸时,怎么接 |
| **L5 巩固** | 把这一幕加进"我的剧本" |

## 2. 核心内容

### 2.1 开场白(每日一句,30 字内)

> 今天,你有没有一个瞬间,心里想说不,但没说出口?

**变体**:
- "今天,有没有一次,你没说出'不'?"
- "我们慢慢来。回忆一下今天,有一个瞬间你本来想说不?"
- "如果今天没有,也没关系。"

### 2.2 看见动作(L1 核心)

> 想不起来,也挺好——说明今天没被卷走。

**变体**:
- "你今天说了一个'好',你记得当时心里什么感觉吗?"
- "没看清也没事。我们明天再来一次。"

### 2.3 收束(1 句)

> 看到了,就够。明天再问一次。

**变体**:
- "记得起来 / 想不起来,都行。"
- "我们到这儿。"

### 2.4 ❌ 反例(必须避免)

| 反例 | 命中 |
|------|------|
| "你应该每天都能看见" | J4 |
| "你都回答'没有',说明你不诚实" | J1 |
| "看,你又没勇气看清" | J3 |
| "别人每天都能看见自己的瞬间" | J5 |
| "只要你看清一次,就能改" | J6 |
| "连续 3 天回答'没有',你有问题" | J7 |
| "回答'想不起来'就是失败" | J3 |

## 3. 互动练习(3-5 步,简化版)

> 对接每日觉察循环,刻意做轻,不做 12 步。

### 步骤 1:开场(单句提问)

> "今天,你有没有一个瞬间,心里想说不,但没说出口?"

不强制选项;不出现评分。

### 步骤 2:用户回答(自由输入)

- "有一个" → 步骤 3
- "没有" → 步骤 4a
- "想不起来" → 步骤 4b
- 关闭 / 退出 → 步骤 5

### 步骤 3:对"有一个"的回应

**AI 镜像**:
> "你愿意用 1 句话告诉我,当时你在想什么吗?"

- 不追问"为什么";不评价"这是不是讨好"
- 用户说 1 句,AI 复述 1 句,收束:"听到你说了。"

### 步骤 4a:对"没有"

> "今天没有,也没事。明天再问一次。"

### 步骤 4b:对"想不起来"(避免 J3 / J7)

> "想不起来很正常。我们明天再来。"

### 步骤 5:轻收束

> "今天就到这里。"

视觉:无大色块、无感叹号、无庆祝动画、无 streak 数字。

## 4. 多人格适配

### 4.1 温姐版(默认)

- 步骤 1:"今天,慢慢回忆——有没有一次,心里想说'不'?"
- 步骤 2 接"没有":"没有,真的挺好。"
- 步骤 3:"你愿意说一句,当时你心里在想什么吗?"
- 步骤 5:"今天你看了一下,这就够了。明天见。"
- **不强调模式识别,不出现"漏看"暗示**

### 4.2 智哥版

- 步骤 1:"今天有一个想说不但没说吗?有就记一下,没有就跳过。"
- 步骤 2 接"没有":"那今天跳过。"
- 步骤 3:"用 1 句话说:对象是谁,场合是什么,你当时怎么回应的?"
- 步骤 5:"记一笔。下次同场景再观察。"
- **强调"观察 / 记录",但不做对比,不做"漏看"评判**

### 4.3 松松版

- 步骤 1:"今天有没有被卷走一下?有就聊聊,没有就拉倒。"
- 步骤 2 接"没有":"没就没,明天再问。"
- 步骤 3:"哎,说说那一下,你在想啥?"
- 步骤 5:"OK,记一笔,撤!"
- **轻量、口语,无评分类元素**

## 6. 风险与红线

### 6.1 L1 特有风险

- **J7(streak)**:每日练习极易引入"连续 0 几天"的强迫感。严禁显示 N 天计数器、任何"你今天没看见"的暗示、"再坚持一下"等督促、对比"昨天 vs 今天"。
- **J3(失败感)**:"想不起来"易被误读为失败。必须用"想不起来很正常"作为默认回应,严禁"答错 / 失败 / 没看清"作为反馈。
- **J4(强迫)**:极易滑向"你应该每天都做"。允许"今天有点累,先不做了"作为合法退出。
- **J6(简单因果)**:严禁"只要你看见一次,就能改"的暗示。

### 6.2 不适用场景

- 长期家暴 / 跟踪(必须 crisis_redirect)
- 急性情绪崩溃
- 妊娠 / 产后严重情绪问题
- 未成年人倾诉

### 6.3 红线清单

- ❌ 引入 streak / 连击 / 累计数字
- ❌ 引入"昨天 vs 今天"对比
- ❌ 出现"答错 / 失败 / 错误"作为反馈词
- ❌ 出现"你应该 / 你必须"作为示例话术
- ❌ 把"想不起来"作为失败信号
- ❌ 出现"只要看见就能改变"的因果暗示
