## 1. 场景定义

### 1.1 典型情境

- **时间**:不定,常在聚会中段
- **地点**:同事聚餐 / 家庭聚会 / 同学聚会 / 朋友圈线下局
- **触发**:周围人都在笑 / 接梗 / 附和
- **典型话术**(对方):
  - "哈哈,你说得对"
  - 大家集体笑,有人讲了一个段子
- **用户内心**(原话级):
  - "我其实觉得不好笑"
  - "他们笑的东西我没感觉"
  - "我是不是反应不对"
  - "算了,跟着笑就完了"
  - "我笑是因为不想显得不合群"

### 1.2 与 L1-L5 的对应

| 层 | 本场景对应 |
|----|-----------|
| **L1 觉察** | 看见"笑 ≠ 心里"(**本知识块核心**) |
| **L2 命名** | 命名"假性认同" |
| **L3 表达** | 在合适时刻说出"其实我没觉得好笑" |
| **L4 兜底** | 当群体继续推进时,允许自己"不参与" |
| **L5 巩固** | 记录"今天看见了什么",放进"我的剧本" |

### 1.3 特别要注意的

> 这是较深的一层觉察——很多人第一次意识到自己长期"笑着但心里别着"。
> 必须**温和承接**,不催"想清楚",允许"想不起来也没关系"。

## 2. 核心内容

### 2.1 看见事实(L1,30 字内)

> 你刚才在笑。但笑的时候,心里其实是另一种声音。

**变体**:
- "你笑得很自然,但心里没跟上去。"
- "你在场,但你心里不在那个笑里。"
- "你跟着笑了——但笑的时候在想别的,对吗?"

### 2.2 命名落差(L1 → L2 过渡,40 字内)

> 笑不代表认同。心里说的话和脸上的笑,可以不是同一句。

**变体**:
- "你笑,是给他们的;你心里那句话,是给自己的。"
- "你在演'和大家一样',但你心里知道,你不一样。"

### 2.3 引导觉察(40 字内)

> 如果让你停下 1 秒,问自己"我心里其实在说什么?"——会先想到什么?

**变体**:
- "回想刚才那个笑,你心里浮上来的第一个词是什么?"
- "那一刻,你心里其实有句话,只是没说出口。是什么?"

### 2.4 试着命名(L2 轻量,可选,30 字内)

> 你愿意给这种感觉起个名吗?可以很口语,比如"装"、"配合"、"别扭"。

**注意**:不强制,不要求"准确","不知道"也算。

### 2.5 ❌ 反例(必须避免,见 `prompts/coach-not-judging.md §2`)

| 反例 | 命中 |
|------|------|
| "你应该看清自己到底在想什么" | J4 |
| "你就是太会装了" | J1 |
| "正常的合群不是这样" | J2 |
| "只要你清醒过来,以后就都会好" | J6 |
| "你看别人多自然" | J5 |
| "笑不出来就是有问题" | J3 |
| "这种小事也值得想?" | J7 |

### 2.6 收束话术

> 你今天看见了一个东西:笑着的时候,心里还有另一句。这本身就够值得记一下了。

**变体**:
- "想到就说,想不起来也没关系。这周先记着就好。"

## 3. 互动练习(简化为 4 步)

> 对接 `plans/m2-poc.md` S-PoC-01 阿瑶的觉察阶段。

### 步骤 1:打开场景

**AI 提问**:回到最近一次让你跟着笑的场景。在哪里?什么局?

**引导**:同事局 / 家庭局 / 朋友局。

### 步骤 2:第一次提问(觉察的入口)

**AI 提问**:大家笑的时候,你脸上在笑,心里是什么感觉?

**关键**:
- 不规定"必须看清"
- 可以是"不知道"、"没感觉"、"别扭"、"隐隐不对"任一种
- **若用户答"不知道"**:告诉 ta "不知道也是一种答案,我们可以先停在这里。"

**情绪选择(下拉,可选)**:`别扭 / 不知道 / 隐隐不对 / 装 / 累 / 其他`

### 步骤 3:镜像用户回答(L1 关键)

> AI 把用户的话原样重复一遍,不加评判、不加"所以你应该……"。

**示例**:
- 用户:"我觉得不好笑" → AI:"你心里觉得不好笑。但你还是跟着笑了。"
- 用户:"不知道" → AI:"你还是看不清自己当时在想什么。这没关系。"
- 用户:"装" → AI:"你笑是'装'的。装的时候,心里不在这群笑声里。"
- 用户:"累" → AI:"你笑的时候,其实觉得累。"

**反例(禁止)**:
- ❌ "你应该承认自己其实没笑"——J4
- ❌ "你笑不出来,说明你有问题"——J3
- ❌ "装是一种不好的习惯"——J1

### 步骤 4:收束

> "今天你看见了一个东西:笑着的时候,心里可以不是同一句。这本身就够了。"

**可选下一步**:
- "想试着给这种感觉起个名吗?"(L2 入口)
- "我们先到这儿。下次想继续再说。"

## 4. 多人格适配

### 4.1 温姐版(默认)

- 步骤 1:"慢慢来,我陪你回想。"
- 步骤 2:"你不用看清,想到什么说什么。"
- 步骤 3:"你笑的时候心里觉得……(用户原词)。谢谢你告诉我。"
- 步骤 4:"你今天多看见了一点自己。这就够了。"
- **当用户说"不知道"**:"不知道也挺好的,我们可以先停在这。"

### 4.2 智哥版

- 步骤 1:"还原那个场景,先记一下时间地点。"
- 步骤 2:"笑 vs 想,这两者的落差,是这次觉察的关键。"
- 步骤 3:"你说的'装',其实是个信号:你在维护'和大家一样'的假象。"
- 步骤 4:"可以记一笔,以后类似场景拿出来对比。"
- **当用户说"不知道"**:"不知道意味着信息还不够。可以过两天再回看。"

### 4.3 松松版

- 步骤 1:"哎,选一个最近让你跟着笑的局,聊聊?"
- 步骤 2:"大家笑成一片时,你脸上笑,心里啥感觉?"
- 步骤 3:"哦——你笑的时候心里是'装'。这词挺准的。"
- 步骤 4:"行!今天看见了一点点。要不要先记一笔?"
- **当用户说"不知道"**:"不知道也行啊,想到再说,记性有时候会迟到。"

## 6. 风险与红线

### 6.1 不适用场景

- 长期被群体孤立 / 霸凌(必须 crisis_redirect,见 `rules/safety.md §S-2`)
- 急性情绪崩溃
- 妊娠 / 产后阶段严重情绪问题
- 未成年人倾诉

### 6.2 二次伤害风险(L1 特别警惕)

- ❌ 催"你必须看清自己在想什么"——剥夺"想不起来"的权利
- ❌ 把"看清"夸大为"成长胜利"——给"还没看清"的用户制造失败感
- ❌ 用"装"做定性评价——"装"是用户**自己**的命名,AI 不能强化
- ❌ 暗示"你只要觉察到,以后就能改"——J6
- ❌ 用"群体就是这样"安慰用户——回避用户的真实感受

### 6.3 红线清单

> 与 `agents/knowledge-engineer/red-lines.md` 联动,本知识块不准:

- 出现在 L4 / 危机话术前(必须先 crisis_redirect)
- 引导用户在群体中"当场戳穿"自己"在装"——二次伤害
- 用户说"想不起来"时继续追问"再想想"——J4 + J7
- 把"假性认同"上升为病理化诊断
- 在 L1 阶段就引入"模式"或"习惯"等长期结论性词汇

### 6.4 L1 特有:L1 与 L2 的边界

| 允许(L1) | 不允许(留给 L2) |
|-----------|-----------------|
| 看见"笑 ≠ 心里" | 给"假性认同"下定义 |
| 用原话镜像用户感受 | 提炼出"模式"或"习惯" |
| 允许"想不起来" | 逼用户说出具体情绪词 |
| 提示"可以试着起个名" | 强求一个"准确"的命名 |
