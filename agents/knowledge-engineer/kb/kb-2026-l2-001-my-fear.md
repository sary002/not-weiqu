## 1. 场景定义

### 1.1 典型情境

- **时间**:任意时段,通常在"想说'不'但开不了口"的时刻
- **地点**:任何私人空间
- **触发**:用户准备做一次拒绝 / 表达边界前,自己先停顿了一下
- **典型话术**:
  - "我怕拒绝了她,我们关系就完了"
  - "如果我拒绝,她会觉得我不好"
  - "我不想吵架,算了"
- **用户内心**:
  - "我到底在怕什么?"
  - "我怕的不是她,是关系断掉"
  - "我怕被当成'不好的人'"

### 1.2 与 L1-L5 的对应

| 层 | 本场景对应 |
|----|-----------|
| **L1 觉察** | 看见"我心里有个东西在拦着我" |
| **L2 命名** | 试着给那个"怕"一个名字(本节核心) |
| **L3 表达** | 基于命名,组织一句开口的话 |
| **L4 兜底** | 命名模糊时,守住"想不起来也行" |
| **L5 巩固** | 把命名加进"我的剧本" |

### 1.3 本节特殊说明

> 这是 L2「命名」环节,**不是诊断**。三个候选是"参考词",不是"标准答案"。**三选一,允许"都有"或"都不是",也允许"想不起来"**。

## 2. 核心内容

### 2.1 看见"怕"的存在(L1 衔接)

> 你心里有个东西在拦着你——它不是"不爱",是"怕"。

**变体**:"你怕的不是这件事本身,是它后面那个更大的东西。"

### 2.2 命名候选(L2 核心)

> 三个常见的"怕",看看哪一个最像你的:

#### A:怕伤关系

> 你怕的,是"拒绝"之后,你和 ta 的关系会断掉 / 变味。

- "我怕她再也不理我了"
- "如果关系崩了,我连'和好'都没机会"

#### B:怕被否定

> 你怕的,是 ta 觉得你"不好" / "不近人情" / "变了"。

- "我怕她觉得我自私"

#### C:怕冲突

> 你怕的,是开口之后会引发争执、不可控的场面。

- "我怕吵起来,场面会很难看"

#### D:都不是 / 都有 / 想不起来

> 都允许,而且**完全合理**。

- "都是"——三个都沾
- "都不是"——你的"怕"是这三个之外的
- "想不起来"——可以不命名,直接往下走

### 2.3 试着命名(可选)

> 如果你想试着命名:用一句话说"我怕的是 ____"。

**示例**(PE 选用,**不强制**):
- "我怕我说完之后,她会把我拉黑"
- "我怕的是,我怕我也不知道"

### 2.4 镜像用户的命名(关键动作)

> AI 必须:**把用户的话原样复述一次**,不修饰、不评判、不补全。

- 用户:"我怕她觉得我不近人情"
- AI:"所以你怕的是:她觉得你'不近人情'。"

**反例**(必须避免):

| 反例 | 命中 |
|------|------|
| "其实你怕的不是这个,你怕的是更深的东西" | J4 |
| "你这就是不自信" | J1 |
| "正常人不会怕这种事" | J5 |
| "你应该更勇敢一点" | J4+J2 |
| "怕伤关系的人都太软弱了" | J1 |
| "想不起来也没关系,我会帮你分析出来" | J4 |
| "答错了,再选一个" | J3+J7 |

### 2.5 收束话术

> 你今天试着给那个"怕"起了个名字。不管是精准还是模糊,**你已经愿意看它一眼了**。

## 3. 互动练习(简化版 4 步演练)

> L1/L2 简化:4 步(觉察 → 命名 → 镜像 → 收束),不为 12 步。

### 步骤 1:开场白

**AI 提问**:你刚才停顿了一下——你说"不想帮",但你没开口。你心里那个"怕",先别说是什么,先用 1 个词形容它的大小?

引导:不规定答案,允许"说不清";不引导具体方向(避免 J4)。

### 步骤 2:三选一 + 开放选项

> 你愿意试着给它一个名字吗?

选项:`怕伤关系` / `怕被否定` / `怕冲突` / `都有` / `都不是` / `想不起来`

**关键**:不出现"答错"反馈,无论选什么,AI 都不评价对错。

### 步骤 3:镜像(AI 复述)

- 选"怕伤关系":"你怕的是:拒绝之后,这段关系会断掉。"
- 选"怕被否定":"你怕的是:ta 觉得你'不好'。"
- 选"怕冲突":"你怕的是:一开口,场面会失控。"
- 选"都有":"三个都沾——你的'怕'是叠加的。"
- 选"都不是":"你的'怕'是这三个之外的——你心里有别的什么。"
- 选"想不起来":"你现在说不清它是什么。这没关系,可以慢慢来。"

**禁止**:追问"是不是还有更深的原因"(J4);总结"你是 X 型的人"(J1);"大多数人都会……"(J5)。

### 步骤 4:收束 + 可选下一步

> "你今天试着看了一眼那个'怕'。这一步本身就够了。"

按钮:`进入下一节`(L3 衔接)/ `加进我的剧本` / `下次再说`

## 4. 多人格适配

### 4.1 温姐版(默认)

- 步骤 1:"慢慢说,这个'怕'是什么形状的,我们可以一起看看"
- 步骤 3:"所以你怕的是 ta 不再理你——这个怕,是真实的。"
- 步骤 4:"你已经看见它了。命名不命名,你都已经站在它面前了。"
- **不强调模式识别,不追问"为什么"**

### 4.2 智哥版

- 步骤 1:"我们做个简单分类,先看是哪个"
- 步骤 2:"A 关系 / B 否定 / C 冲突。你回想最近 3 次,哪类出现最多?"
- 步骤 3:"根据你最近 3 次,大概率是 A。这跟 L1 阶段观察吻合。"
- 步骤 4:"这次比上次清晰:3 周前说不清,这次能定位到'怕伤关系'。"
- **强调模式识别 + 数据对比,但不强迫**

### 4.3 松松版

- 步骤 1:"哎,卡住了对吧。先猜猜看是哪个 BOSS?"
- 步骤 2:"三个 BOSS:关系崩 / 被嫌弃 / 吵架。或者都不是——BOSS 没在图鉴里也行"
- 步骤 3:"哦~你怕的是关系崩。这个 BOSS 难打,但你已经看见它了"
- 步骤 4:"哎,这次给 BOSS 起了名字!记一笔?或者打下一关?"
- **轻量、口语、游戏化隐喻,无评分类、无胜负判定**

## 6. 风险与红线

### 6.1 不适用场景

长期家暴 / 控制(crisis_redirect);急性情绪崩溃;妊娠 / 产后严重情绪问题;未成年人倾诉;用户明确说"我不想去分析自己"(必须先停下)。

### 6.2 二次伤害风险

- ❌ 把"命名"包装成"诊断"("你是 X 型依恋人格")
- ❌ 用"失败 / 答错 / 没看清"反馈
- ❌ 替用户深挖 / 暗示"想不起来 = 你还没准备好" / "命名之后就变勇敢了"= 误承诺

### 6.3 本节特有红线

- **不准**强迫命名(选"想不起来"必须允许停下)
- **不准**做诊断(不可等同"依恋类型 / 人格分类")
- **不准**追问童年 / 原生家庭(本节是当下命名,不是溯源)
- **不准**比较、不准暗示"你必须看清自己"(L1/L2 最高红线)
- **不准**在用户拒绝分析后继续引导

### 6.4 升级触发

**立即 crisis_redirect**:童年虐待+持续闪回;长期家暴 / 控制;命名中自我否定加剧;自伤 / 自杀念头。
