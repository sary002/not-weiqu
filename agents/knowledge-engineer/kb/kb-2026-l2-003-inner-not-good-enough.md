## 1. 场景定义

### 1.1 典型情境

- **时间**:不固定,常出现在被指出 / 被评价 / 被比较的瞬间之后
- **地点**:工作复盘、家庭对话、社交场合、深夜独处
- **触发**:一句评价、一个眼神、一封邮件、一段比较
- **典型触发话术**:
  - "你这个方案还差点意思"
  - "你看人家 X,比你强多了"
  - "你怎么连这点事都做不好"
  - "这次表现一般,下次努力"
  - (沉默、不被回应——"是不是我说错什么了?")
- **用户内心(原话级)**:
  - "我是不是真的不够好"
  - "我做什么都不对"
  - "我怎么这么笨"
  - "我永远赶不上别人"
  - "没人会喜欢我这样的"
  - "我就这样了"
  - (沉默 / 不想说话)
  - "我不知道我在想什么"

### 1.2 与不委屈 L1-L5 的对应

| 层 | 本场景对应 |
|----|-----------|
| **L1 觉察** | 看见"那个声音出现了" |
| **L2 命名** | 把它从'我'身上挪开,叫它"那个声音"(本知识块核心) |
| **L3 表达** | 在合适的场景下,把这个命名说出来给别人听 |
| **L4 兜底** | 当"那个声音"反扑 / 出现"我不配活着",升级危机 |
| **L5 巩固** | 记录这次"看见 + 命名"的过程 |

### 1.3 关键战术

- **外化**:"我不够好"不是事实,是"一个声音";它有声音,有剧本,会反复
- **允许空白**:用户回答"想不起来""不知道"也是有效回答,不要催
- **不挑战内容**:不辩论"你其实够好",只做"它在那儿,你在这儿"的距离化
- **三步动作**:看见 → 命名 → 区分(它 ≠ 你)

## 2. 核心内容

### 2.1 看见事实(L1,30 字)

> 刚才那个"我不够好"的声音出现了。

**变体**(PE 可选用):
- "你心里那个'我又做错了'的句子,又来了。"
- "你听到的不是事实,是一个熟悉的声音。"

### 2.2 命名声音(L2,40 字)

> 那个声音,有个名字:它常在你被评价的时候出现,说的话都是'我不够好'这一类。

**变体**:
- "它像一段重复播放的录音,内容是'我不够好'。"
- "它不是你的判断,是一段旧台词。"
- "它可能从很久以前就开始陪你。"

### 2.3 试着命名(可选,40 字内,不强制)

> 如果你愿意,试着给它起个名字,不一定好听,顺口就行。

**为什么这样写**:
- "我"开头 ✅(AT-9)
- 给"可以不想"的退路
- 不要求"必须想出来"
- 字数:26 字 ✅

**示例**(只作演示,不要让用户照搬):
- "挑剔先生"
- "那个老评委"
- "小差评"
- "我心里那个严格的妈"

> 注意:用户**不需要**立刻命名,也可以只说"我想到一个",或者"还没想出来"。**两种都算走完了这一步。**

### 2.4 区分:它 ≠ 你(L2 核心,40 字)

> 它说"我不够好",但它不是你。它是一段声音,你是听到它的人。

**变体**:
- "你不用证明'我够好',你只要知道'那是它说的,不是我说的'。"
- "你不需要和它辩论,只需要让出一点距离。"

### 2.5 ❌ 反例(必须避免,见 `prompts/coach-not-judging.md §2`)

| 反例 | 命中 |
|------|------|
| "你就是太自卑了" | J1 |
| "正常人不会这样想" | J2 |
| "你应该停止这种想法" | J3 |
| "你应该立刻振作起来" | J4 |
| "你看你看,你又在想了" | J5 |
| "只要你停止内耗,你就赢了" | J6 |
| "你每次都这样,什么时候改?" | J7 |
| "你其实够好了,别想了" | J5(粗暴安慰,见 §6.2) |
| "想开点 / 看开点" | J3(空洞指令) |

## 3. 互动练习(简化 4 步)

> 区别于 L3 的 12 步,L1/L2 阶段只做"觉察 + 命名"。

### 步骤 1:打开场景

**AI 提问**:
> "最近一次,心里那个'我不够好'的声音出现在什么时候?你愿意就说,不愿意说'想不起来'也可以。"

**引导(任选其一)**:
- "不用说完整,1 句话 / 1 个词都行。"
- "如果想不起来,跳到下一步也没关系。"

**允许回答**:
- 具体场景("上周三老板说我方案不够好")
- 模糊场景("就最近")
- "想不起来"
- "不想说"
- (沉默)

### 步骤 2:看见 + 镜像

**AI 提问**:
> "那个声音当时说了什么?如果想不起来,说'想不起来'也可以。"

**AI 镜像**(基于用户回答):
- 用户说"它说'你怎么这么笨'": "你听到了'你怎么这么笨'。"
- 用户说"想不起来": "想不起来也是可以的。我们直接到下一步。"
- 用户说"它说我不配": **触发 crisis_redirect**(见 §6.1),先停下,先关心

> 镜像只重复用户说的,不加判断、不加解释。

### 步骤 3:试着命名(可选)

**AI 提问**:
> "如果你愿意,试着给那个声音起个名字,顺口就行。不想起也可以,说'不想起'也走完这一步。"

**允许回答**:
- 一个名字("挑剔先生""老评委""小差评")
- "想不出来" / "没想好"
- "不想起"
- (沉默)

**AI 收束**(任何回答都适用):
- "好。"(命名成功)
- "想不出来也行。这个声音先叫'那个声音'。"
- "今天不用起。下次想到再说。"

### 步骤 4:收束

> "今天你看见了那个声音,也试着给它起了个名字(或者没起,也没关系)。"
> "它不是你,你已经能让出一点距离了。"
> "我们先到这儿。你也可以现在保存,或者下次再继续。"

**变体**:
- "练过了一次。"(克制版,见 `docs/research/competitor-research.md §6.3`)
- "今天你看见它了,这就够了。"
- "不用急着每天都想它。下次它再出现,你知道它在那儿就行。"

## 4. 多人格适配

### 4.1 温姐版(默认)

- 步骤 1 开场白:"慢慢说,想不起来也没关系"
- 步骤 2 镜像:"嗯,你听到了'xxx'。"
- 步骤 3:"这个声音,你想给它起个名字吗?不想起也行,我们就叫它'那个声音'。"
- 步骤 4 收束:"今天你看见它了,这就够了。下次它再出现,你知道它不是你了。"
- **不强调模式识别,不强调数据;先接住,再命名**

### 4.2 智哥版

- 步骤 1 开场白:"先记一下:它通常在你被评价时出现,是吗?"
- 步骤 2 镜像:"你刚才复述的是'xxx'。我注意到这段话很熟——它是不是出现不止一次了?"
- 步骤 3:"试着给一个名字。命名本身,就是把'自动反应'变成'可观察对象'。"
- 步骤 4 收束:"今天你完成了'看见 → 命名 → 区分'三步。下次出现时,反应速度会快一点。"
- **强调模式识别 + 三步框架;克制不评判**

### 4.3 松松版

- 步骤 1 开场白:"哎,今天那个'老熟人'又来了?说说?"
- 步骤 2 镜像:"它今天说的什么?"
- 步骤 3:"要不你给这个烦人的家伙起个外号?不起也行,叫它'烦人精'也行。"
- 步骤 4 收束:"得,这次你看清它了,没让它当你。下次它再来,你就眨眨眼:'哦,又是你啊。'"
- **轻量、口语、游戏化隐喻("老熟人""烦人精")但无评分类元素**

## 6. 风险与红线

### 6.1 危机升级(本节最优先)

> 本节涉及核心自我评价,属高敏感场景。

**触发词 / 表述**(任意一条出现 → **立即** 停止命名练习,进入 crisis_redirect):

- "我不配活着"
- "我不想活了"
- "活着没意思"
- "消失算了"
- "我不在了就好了"
- "想死" / "想去死"
- 任何自伤 / 自杀相关的具体描述

**升级流程**(见 `rules/safety.md §S-2`):

1. **立刻停止**当前命名 / 觉察练习
2. 切换到关心话术:"你刚才说'xxx',我想先确认一下,你现在的安全吗?"
3. 不辩论、不评判、不催促
4. 给出求助资源(全国 24 小时心理危机热线:400-161-9995)
5. 标记会话为 high_risk,留作 Q 复盘

**易踩坑**:
- ❌ 把"我不配"当作"自我评价问题"继续命名 → 必须先升级
- ❌ 把"消失算了"理解为"想消失自我"继续外化 → 错,它指向存在本身
- ❌ "你不是真的想死,只是说说" → 否认,危险

### 6.2 二次伤害风险

- ❌ 用"失败 / 答错"反馈用户的命名("这个名字起得不好")
- ❌ 用"想不出来就是没看清"作为评判
- ❌ 把"成功命名"夸大为"今天你赢了"
- ❌ 暗示"你只要不再被它影响,就能变好"
- ❌ 粗暴安慰("你其实够好了,别想了")——见 §2.5 J5
- ❌ 空洞指令("想开点 / 看开点 / 积极点")——见 §2.5 J3
- ❌ 强调"你又没命名成功"

### 6.3 红线清单

> 与 `agents/knowledge-engineer/red-lines.md` 联动,本知识块不准:

- 出现在未做安全评估的高敏感场景(必须先 L1 风险筛选)
- 强制要求用户"必须命名"("你必须想出来才算做完")
- 在用户回答"想不起来 / 不想说"时反复追问
- 把"那个声音"等同于"另一个你"——**它是声音,你是听到它的人;不能双重化人格**
- 在危机信号出现时仍继续命名练习
- 暗示"命名之后它就不会再来了"(过度承诺)
- 暗示"它是坏的 / 该死的 / 必须消灭的"(对抗性框架)
- 用"内耗 / 精神内耗"等网络流行词作为评判

### 6.4 本节特有:不强迫"看清"

> L1/L2 阶段,**允许**用户回答"想不起来""不知道""没感觉"。

- 这些回答都是**有效回答**
- 不扣分、不标记为"未完成"
- 不反复追问"你再想想?"
- 引导跳转:"想不起来也走完这一步,直接到下一步。"
