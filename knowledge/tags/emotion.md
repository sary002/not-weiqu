# tags · emotion/*

> 情绪命名标签。用于标记用户主导情绪与副情绪。

| 二级 | 含义 | 出现场景 |
| --- | --- | --- |
| `emotion/委屈` | 感到被不公平对待但说不出口 | 职场、家庭、关系 |
| `emotion/愤怒` | 感到被侵犯，需要表达 | 边界被越界 |
| `emotion/内疚` | 觉得"应该"满足对方 | 讨好 / 隐忍 |
| `emotion/焦虑` | 对未来结果过度担心 | 决策 / 表达 |
| `emotion/孤独` | 感到没人理解 | 关系 / 家庭 |
| `emotion/恐惧` | 担心后果 | 表达 / 拒绝 |
| `emotion/羞耻` | 觉得"我不好" | 自我价值低 |
| `emotion/疲惫` | 长期消耗 | 隐性边界越界 |
| `emotion/无力` | 觉得改变不了 | 长期压抑 |
| `emotion/感激` | 表达谢意（积极） | 关系确认 |

## 标注规则
- 单条切片可标 1~3 个 emotion/*
- 主导情绪放第一位
- 不与 `selfworth/*` 冲突：emotion 是即时情绪，selfworth 是状态
