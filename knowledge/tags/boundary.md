# tags · boundary/*

> 边界类型标签。用于标记被越界的具体边界。

| 二级 | 含义 | 典型话术 |
| --- | --- | --- |
| `boundary/time` | 时间边界（私人时间被占用） | "周末让我加班" |
| `boundary/space` | 空间边界（个人空间被侵犯） | "你必须跟我住" |
| `boundary/emotional` | 情绪边界（被迫承接对方情绪） | "你必须原谅我" |
| `boundary/decision` | 决策边界（人生决定被代替） | "你应该结婚" |
| `boundary/financial` | 经济边界（钱被控制） | "工资上交" |
| `boundary/physical` | 身体边界（触碰 / 性） | "不同意就分手" |
| `boundary/intellectual` | 思想边界（观点被强制） | "你必须同意我" |

## 与五层模型挂靠
- L1 觉察：识别哪一类边界被侵犯
- L2 命名：说清被侵犯的具体边界
- L3 表达：就具体边界说"我可以 / 我不可以"
- L4 兜底：被回绝后自我安抚
- L5 巩固：守住边界成为习惯
