# tags · pua/*

> PUA 模式标签。用于识别和命名操控模式。

| 二级 | 含义 | 典型话术 |
| --- | --- | --- |
| `pua/责任转移` | 把 A 的责任转嫁给 B | "都是你不配合" |
| `pua/比较打压` | 用别人贬低对方 | "你看 XX 多优秀" |
| `pua/煤气灯` | 让对方怀疑自己的记忆 / 感受 | "我没说过那话" |
| `pua/打击自信` | 系统性贬低对方能力 | "你能做什么" |
| `pua/制造依赖` | 让对方离不开自己 | "离开我你什么都不是" |
| `pua/情感操控` | 用情绪勒索 | "你不做就是不爱我" |

## 与 gaslighting 的区别
- pua 偏"策略"（主动）
- gaslighting 偏"否认现实"（被动）
- 两者可同时存在
